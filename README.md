# ICC T20 World Cup Analysis (2007–2021)

## Project Overview
This project analyzes ICC T20 World Cup match data from 2007 to 2021 to uncover trends in team performance, player impact, and tournament structure using Python.

## Dataset
The dataset contains match information from multiple ICC T20 World Cup editions including:

- Match details
- Teams playing
- Venue
- Match result
- Player of the Match
- Tournament stage

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- VS Code

## Analysis Performed
The project explores several key aspects of the tournament:

1. Number of matches played in each World Cup year
2. Most successful teams in tournament history
3. Players with the most Player of the Match awards
4. Venues hosting the most matches
5. Distribution of matches by tournament stage

## Key Insights
- The number of matches has increased in later tournaments.
- Some teams consistently dominate the competition.
- A few players stand out for their repeated match-winning performances.
- Certain stadiums host a large number of matches.
- The majority of matches take place during the group stage.

## Conclusion
The analysis highlights patterns in ICC T20 World Cup history and demonstrates how data analytics can provide meaningful insights into sports performance.
